FAQ

Linux Mint とは?

Linux Mint はデスクトップパソコンで使うことを想定し、設計された Linux ベースのフリーな OS です。

Linux Mint は、企業に適していますか?

適しています。 Linux Mint のリリースは 6 ヶ月毎ですが、それぞれのリリースは 9 ヶ月サポートされます。 また2年ごとに LTS (Long-Term Support, 長期サポート) 版がリリースされ、 5 年間のサポートと次回 LTS リリースへのアップグレードが保証されています。

Linux Mint は個人に適していますか?

明らかに適しています。私たちは Linux は出回っている OS のなかで良い物であると思っています。 Linux ディストリビューションは 300 以上の種類がありますが、私たちはデスクトップパソコンの最良のソリューションとなるべく努めています。 他に優れたデスクトップ OS やディストリビューションが出ていますので最良のディストリビューションを提供するのは大変です。 私たちは自分たちのデスクトップの品質やたくさんの偉大な理念に確信を持っています。 例えばあなたのパソコンが 512MB 以上の RAM メモリーがあるとすれば、 Linux Mint を試す価値は十分にあると考えます。 私たちはユーザーの経験を積極的に聞きたいのです。 そしてユーザーの提案を尊重します。 ですから、ぜひ私たちにあなたの意見をお寄せ下さい。

なぜ Linux Mint はこれほど多くのエディションがあるのですか?

私たちは違った役割のためにデザインされた、さまざまなバージョンの Linux Mint を提供しております:

  • The Main Edition (メイン版) は私たちの最も良いエディションです。 これはパソコンのマルチメディアを完全にサポートしています。 つまりあなたは MP3 を聞けますし、DVD も見れて、 Flash が必要なホームページを見ることができます。そしてそれはインストール直後からすぐに行えます。

  • No Codecs (コーデック無し) は日本やアメリカなどのソフトウェアに特許権がある国で自由に配布出来るようにデザインされています。 これには完全なマルチメディアのサポートはありません。

  • The KDE Community Edition (KDE コミュニティ版) は、人気のあり有名な KDE デスクトップで構築されたバージョンの Linux Mint です。

  • The Xfce Community Edition (Xfce コミュニティ版) は、軽いモジュール式の Xfce デスクトップで構築されています。

Linux Mint には商用のドライバーが含まれているのですか?

含まれていません。もし含んでいると法的に (GPL を侵害するので) 、あるいは倫理的に (ユーザーがコードをカーネルにリンクするといった汚い行為をするなど) 問題があります。

プロプライエタリなソフトウェアは?

私たちはオープンソースの哲学を信じており、すべての製品のソースコードを公開してします。 私たちはフリーソフトウェア運動や GPL を支持しています。 しかし優れたアイデアを持っていたり、すばらしいツールを創造したり、ソフトウェアをよりよくするために働いている開発者たちもいます。 いくつかの人は自分のソフトウェアのソースコードを公表してしている人もいるため、私たちにより多くのフリーと柔軟性を与えてくれます。 一方で、自分のソフトウェアをソースコードなしで商用ライセンスでリリースしている人もいます。 これは私たちが求めているフリーを与えてはくれませんが、やはりソフトをよりよくするのに貢献しています。 私たちは一般的なソフトウェアが好きで、フリーソフトウェアはなお一層好きですが、商用ソフトウェアをボイコットしたいとまでは思っていません。
