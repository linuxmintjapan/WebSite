FAQ

Linux Mint とは?

Linux Mint はデスクトップパソコンで使うことを想定し、設計された Linux ベースのフリーな OS です.

Linux Mint は、企業に適していますか?

適しています. Linux Mint のリリースは 6 ヶ月毎ですがそれぞれのリリースは 1 年半サポートされます.また断続的に LTS リリース (Long-Term Support, 長期サポート) がされ、3 年間のサポートと次回 LTS リリースへのアップグレードの保証がされています.

Linux Mint は個人に適していますか?

明らかに適しています.私たちは Linux は出回っている OS のなかで良い物であると思っています。.Linux ディストリビューションは 300 以上のがありますが,私たちはデスクトップパソコンの最良のソリューションとなるべく努めています.他に優れたデスクトップ OS やディストリビューションが出ていますので最良のディストリビューションを提供するのは大変です。私たちは自分たちのデスクトップの品質やたくさんの偉大な理念に確信を持っています.例えばあなたのパソコンが 512MB 以上の RAM メモリーがあるとすれば, Linux Mint を試すといいと思います。私たちはユーザーの経験を積極的に聞きたいです。そしてユーザーの提案を尊重します。ですから私たちにあなたの意見をお寄越し下さい.

なぜ Linux Mint はこれほど多くのエディションがあるのですか?

私たちは違った役割のためにデザインされた、さまざまなバージョンの Linux Mint を提供しております:

  • The Main Edition(メイン版) は私たちの最も良いエディションです.これはパソコンのマルチメディアを完全にサポートしています。つまりあなたは MP3 を聞けますし、DVD も見れて、 Flash が必要なホームページを見ることができます。そしてそれはインストーラ直後にすぐに出来ます.

  • The USA-Japan Edition(日米版) はソフトウェアに特許権がある国で自由に配布出来るようにデザインされています.これには完全なマルチメディアのサポートはありません。

  • The KDE Community Edition(KDE コミュニティ版) は,人気のあり有名な KDF デスクトップで構築されたバージョンの Linux Mint です。

  • The XFCE Community Edition(XFCE コミュニティ版) は,軽いモジュール式の XFCE デスクトップで構築されています.

なぜ Linux Mint には商用のドライバーが含まれているのですか?

含まれていません.もし含んでいると法的 (GPL を侵害するので)に,あるいは倫理的に(ユーザーがコードをカーネルにリンクするといった汚い行為をするなど) 問題があります.

プロプライエタリなソフトウェアは?

私たちはオープンソースの哲学を信じており、すべての製品のソースコードを公開してします。私たちはフリーソフトウェア運動や GPL を支持しています。しかし私は優れたアイデアを持っていたり、すばらしいツールを創造したり、ソフトウェアをよりよくするために働いている開発者たちもいます。いくつかの人は自分のソフトウェアのソースコードを公表してしている人もいるため、私たちにより多くのフリーと柔軟性を与えてくれます。一方で 自分のソフトウェアをソースコードなしで商用ライセンスでリリースしている人もいます。これは私たちが求めているフリーを与えてはくれませんが,やはりソフトをよりよくするのに貢献しています。私たちは一般的なソフトウェアが好きで,フリーソフトウェアはなお一層好きですが,商用ソフトウェアをボイコットしたいとまでは思っていません。
